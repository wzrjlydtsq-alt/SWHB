# 场景导入 - 示例文档

## 使用说明
1. 点击底部栏 ⚡ **自动化操作** → **场景导入**
2. 在弹出面板中：
   - **方式一**：拖拽或选择包含场景标记的 `.txt` 文件
   - **方式二**：直接粘贴文本到输入框，点击「解析场景」
3. 系统按 `场景1`、`场景2`...（或 `场景一`、`场景二`...）自动分段
4. 预览解析结果后，点击「生成 AI 视频节点」或「生成 AI 绘图节点」
5. 画布上自动创建对应数量的节点，每个节点内已填入场景描述文本

---

## 输入文本示例

```
场景1
深夜的码头。月光洒在斑驳的水泥地面，生锈的起重机矗立远处。
海面泛着暗淡波光，几辆黑色轿车停在仓库旁。
空气中弥漫海水和柴油的味道。

场景2
清晨的市局办公室。堆满案卷的办公桌，白板上贴满照片和红色连线。
窗外阳光刺眼，百叶窗帘半遮，台灯暖光亮起。
气氛紧张压抑。

场景3
晚上八点，老城区窄巷内的火锅店。
霓虹灯招牌闪烁"重庆老灶"，店内人声鼎沸热气腾腾。
烟雾缭绕中适合秘密接头。
```

---

## 解析效果

系统会识别出 **3 个场景**：
- **场景1** → 深夜码头描述 → 填入第1个节点的提示词
- **场景2** → 办公室描述 → 填入第2个节点的提示词
- **场景3** → 火锅店描述 → 填入第3个节点的提示词

生成的 AI 绘图/视频节点可直接执行，省去手动复制粘贴的操作。
