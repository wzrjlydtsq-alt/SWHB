# 角色导入 - 示例文档

## 使用说明
1. 点击底部栏 ⚡ **自动化操作** → **角色导入**
2. 在弹出面板中：
   - **方式一**：拖拽或选择包含角色标记的 `.txt` 文件
   - **方式二**：直接粘贴文本到输入框，点击「解析角色」
3. 系统按 `角色1`、`角色2`...（或 `角色一`、`角色二`...）自动分段
4. 预览解析结果后，点击「生成 AI 视频节点」或「生成 AI 绘图节点」
5. 画布上自动创建对应数量的节点，每个节点内已填入角色描述文本

---

## 输入文本示例

```
角色1
张小明，25岁男性，刑警队长。
短发寸头，左脸有刀疤，身材魁梧。
穿着洗得发白的深蓝色风衣。

角色2
林小雨，26岁女性，省厅高材生。
马尾辫，圆脸大眼，外形清秀。
黑色运动外套，行事利落干练。

角色3
陈伟（绰号 "白鹤"），40出头男性，毒枭。
瘦高个，头发一丝不苟，金丝眼镜。
白色西装，嘴角永远挂着微笑。
```

---

## 解析效果

系统会识别出 **3 个角色**：
- **角色1** → 张小明的描述 → 填入第1个节点的提示词
- **角色2** → 林小雨的描述 → 填入第2个节点的提示词
- **角色3** → 陈伟的描述 → 填入第3个节点的提示词

生成的 AI 绘图/视频节点可直接执行，省去手动复制粘贴的操作。
